# Renombrador de Archivos Masivos 🗃️

Script en Python que renombra archivos de forma masiva con un prefijo personalizado.

## Ejemplo
```
foto1.jpg  →  vacaciones_2026_001.jpg
foto2.jpg  →  vacaciones_2026_002.jpg
foto3.jpg  →  vacaciones_2026_003.jpg
```

## Uso
```bash
python renombrador.py
```

1. Ingresa la ruta de la carpeta con los archivos a renombrar
2. Ingresa el prefijo que quieres usar
3. Confirma y listo ✅

## Logs
Cada renombrado queda registrado en `logs/registro.txt` con fecha y hora.
